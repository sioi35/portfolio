개발자 커뮤니티

JSP/Servelt을 활용한 홈페이지 제작

주제 / 기획의도

1.    기존의 커뮤니티 사이트에 대상을 개발자로 특정하여 개발자 전용 기능을 추가한 커뮤니티를 제작하였습니다.

2.    게시판 이용 뿐 아니라, 원하는 스터디를 등록 및 지원 할 수 있다.

3.    IT관련 채용 공고만을 모아 볼 수 있다.

4.    스터디를 할 장소 걱정없이 공간 대여 서비스로 원하는 공간을 편리하게 이용할 수 있다.

사용 기술

기술 분야 상세 내용
View 자주 조회하는 테이블과 조인이 필요한 테이블을 view를 이용해 가상의 테이블로 만들어 둬 재사용성을 높였습니다.
Procedure 컬럼을 삭제/수정하는 등의 조작이 필요한 경우 프로시저를 이용해 변경하고자 하는 컬럼번호와, 컬럼 값을 입력하여 변경을 쉽게 하도록 구현하였습니다.
과정이 종료되면 해당 과정을 수강한 교육생들의 목록을 교육 종료 교육생 테이블로 이동시켜 현 교육생과 종료 교육생을 분리시켜 관리가 편하도록 프로시저를 구현하였습니다.
Function 복잡한 계산을 해야하는 교육생들의 성적 총 합과, 취업 현황 정보를 제공할 때 교육생들의 이름을 익명으로 표시해주기위해 함수를 구현하였습니다.
Trigger 데이터 조작에 자동으로 실행되는 프로시저를 구현하여 교육생들의 출결 신호에 이상 시간(지각, 조퇴, 외출)이 들어오면 이를 처리해 주는 트리거를 구현하였습니다.
Index 테이블에 있는 데이터를 빨리 찾기 위해 index를 구현해 사용하였습니다.

최종 후기

   오직 개발자들만을 위한 커뮤니티 웹 사이트입니다. 커뮤니티 사이트는 많지만 채용공고, 공간 대여 서비스 등을 추가하여 타 사이트와 차별된 사이트를 만들었습니다. 이번 웹 사이트를 제작하며 개발자의 입장에서 사용자의 편의를 높이기 위해 팀원들과 많은 회의를 거치며 명세서를 꼼꼼히 작성하여 구현 단계에서 실수를 줄였습니다. 프론트
